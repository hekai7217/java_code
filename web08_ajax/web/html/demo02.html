XMLHttpRequest的属性:
readyState 属性 :
0：未初始化。对象已创建，未调用open；
1：open方法成功调用，但Send方法未调用；
2：send方法已经调用，尚未开始接受数据；
3：正在接受数据。Http响应头信息已经接受，但尚未接收完成；
4：完成，即响应数据接受完成。

status 属性: 服务器的返回的http的状态码:
200 成功
404 路径找不到 url写错误了
500 服务器的问题

responseText 服务器给回写的数据
onreadystatechange
监听readState 状态触发器, 如果 readyState 属性发生了变化 onreadystatechange 会监听到,执行对应的触发函数
XMLHttpRequest的的方法:
method请求的方法
url 请求地址
asynch 是否异步

open(string method,string url,boolean asynch)
初始化,xhr的组件,
send(content) 向服务器发送数据 ,

SetRequestHeader(String header,String Value)
给服务器发送头信息 ,post 需要发送头信息 (request.setRequestHeader("Content-Type", "application/x-www-form-urlencoded");)
告诉 服务器的数据格式,键值对的格式,一般用于open方法的后面
